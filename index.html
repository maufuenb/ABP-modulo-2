Aplicación de consola

Este proyecto fue desarrollado para el ABP del módulo 3 de JavaScript. La lógica principal corre en la consola del navegador y usa prompt(), alert(), funciones, condicionales, ciclos, arreglos y objetos.

  1. Abrí esta página en el navegador.
  2. Presioná F12 para ver la consola.
  3. Hacé clic en el botón para iniciar la aplicación.
